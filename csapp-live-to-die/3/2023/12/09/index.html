环境配置

我记得其实我第一次学的时候，就是因为配不好环境而没有做，所以我现在写了一个脚本，和构建了一个完整的 docker 环境，让配置环境只需要几行命令就搞定，这样大家就不会因为环境而被劝退了。

首先是 Linux环境 ：

  1. Windows10/11 三步安装wsl2 Ubuntu20.04（任意盘） - 知乎 (zhihu.com)
  2. 虚拟机 安装 Ubuntu20.04，这个比较简单，找一篇博客就行，不过记得用 VMware pro 16，很方便 ssh，如果电脑配置不高，记得 Ubuntu 用 server版本。而不是 desktop

在这两种方式中 WSL 安装配置比较简单，也比较高效，但是有可能会有问题，特别是在 Win11 上。Win10 上使用我没有碰到什么问题，并且安装也比较简单，如果你是Win10 我还是强烈建议使用 WSL的

下面是 CSAPP 环境配置，首先如果你用的是 Ubuntu20.04 及以下，那么你可以直接运行下面的脚本，就可以安装好，当然你也可以使用第二种方法 Docker。 操作都很简单，就几行命令而已，Docker 方法适用性更加广，目前大多数操作系统都支持。
