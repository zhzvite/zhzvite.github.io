DP之背包问题

2023/10/11 学习笔记

本篇主要讲自己学习动态规划时走过的坑,顺便给你们埋坑以背包问题展开对DP(动态规划)的学习

DP-背包的基础入门
#01背包
题意：有n个物品，每个物品的质量为w[i]，每个物品的价值为v[i],背包的容积最多为j，求最大价值

思路分析
设dp[i][j]为考虑i种情况下，体积为j的情况下最大的经济效益。
抓住状态转移方程式d[i][j]=max(d[i-1][j],d[i-1][j-w]+v)
附上代码

for (int i = 1; i <= n; i++) {
    for (int j = 0; j <=m; j++) {
        f[i][j] = f[i - 1][j];
        if (j >= v[i])f[i][j] = max(f[i][j], f[i - 1][j - v[i]] + w[i]);
    }
}cout << f[n][m];`

注意上述背包只利用了i-1和i两个数组下标，故可以对其进行优化，使用滚动数组
注意降维时选用的dp[i-1][j]是在dp[i][j]前面，为防止覆盖，应对二次循环倒序遍历
故代码为:

    for (int i = 1; i <= n; i++) {
        for (int j = m; j >=v[i]; j--) {
            f[j] = max([j], f[j - v[i]] + w[i]);
        }
    }cout << f[m];


#完全背包
题意与01背包类似,但选用的物品有无数件
**思路分析**
没啥区别上代码块
```for (int i = 1; i <= n; i++) {
    for (int j = 0; j <= m; j++) {	
        for (int k = 0;k*w[i]<=j; k++)	{
            if (k * w[i] <= k)dp[i][j] = max(dp[i][j], dp[i - 1][j - k * w[i]] +k * v[i]);
           }
       }     
    }
```
```
**代码优化**
可以通过数学知识运算将内层循环删掉，变成下列代码
```  
    dp[i][j] = dp[i - 1][j];
    if(j>=w[i])dp[i][j] = max(dp[i - 1][j], dp[i ][j - w[i]] + v[i]);//数学上的化简```

如此将dp[i][j]的前一次递归放到了dp[i][j-w]+v,这样再优化成一维时就不用内层循环逆序了
及如此

    for (int j = w[i]; j <= m; j++) {
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);//数学上的化简
    }
}
cout << dp[m] << endl;```
