kmp

本文最后更新于：2023年12月5日 下午

昨天,在CSDN上看见了一篇高质量的blog专讲kmp的,他的讲法比较好懂,但个人觉得他的模板没有y总的简洁,易背诵
先看下y总的模板

#include<iostream>
using namespace std;
const int N = 10010, M = 100010;
int n, m;
char p[N], s[M];//p是模式串,s是模板串
int ne[N];
int main() {
    cin >> n >> p + 1 >> m >> s + 1;//p+1是指从第一位开始存
    //求next的过程,模板串
    for (int i = 2, j = 0; i <= n; i++) {//这里的ne【i】指的是前1-i项构成的序列中前缀和后缀相同的长度
        while (j && p[i] != p[j + 1])j = ne[j];
        if (p[i] == p[j + 1])j++;
        ne[i] = j;
    }
    //kmp 匹配过程
    for (int i = 1, j = 0; i <= m; i++) {
        while (j && s[i] != p[j + 1])j = ne[j];
        if (s[i] == p[j + 1])j++;
        if (j == n) {
            cout << i - n<<" ";
            j = ne[j];//匹配成功
        }
    }
    return 0;
}

y总的模板求模板串的next数组和匹配过程类似,j都要初始化为0,当j>0时才证明已经匹配成功一个字符了,之后再不成功才有j=ne[j],进行退回操作
